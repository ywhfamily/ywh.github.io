Polygon Block Game

2021-02-16

Four Tile Matching Game in One.

This game include FOUR type of tile matching Games. The board consists of sticks, triangles, squares or hexagons. The square board is same as famous 1010 or woody puzzle. But the stick board is rare to see. Each game have Five board size and More block shape types. The game is simple and relaxing. Just drag the block and drop to the empty place of board. when a line(Horizontal, vertical, diagonal) is filled with the blocks, they will disappear.

How to Play

  1. Objective: Complete a line of blocks across the board to remove them. The more removed, The high scores.
  2. Play: Drag the block and drop it on the empty place of board.
  3. Click the icon on the upper right corner to renew blocks, Three chances in a round.
